This website runs on HTTPS but the API URL that you use might be on HTTP. Your browser may not allow AJAX calls from HTTPS based origin to HTTP based target. You'll need to configure your browser to allow AJAX calls to HTTP based URLs in order to proceed.

Steps for unblocking HTTP based API URLs in Google Chrome

Google Chrome
⇢ Hamburger Menu
 ⇢ Settings Menu
  ⇢ Site Settings
   ⇢ Insecure Content
    ⇢ Click Add button of Allow
     ⇢ Add https://kite-open-api.github.io/
